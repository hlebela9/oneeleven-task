OneEleven Task Project
Getting Started

These instructions will help you run the project locally on your machine.

Prerequisites

Node.js
 installed (v16+ recommended)

npm (comes with Node.js)

Installation

Clone the repository

git clone https://github.com/hlebela9/oneeleven-task.git

Navigate to the project folder

cd oneeleven-task

Install dependencies

npm install
Running the Project

Start the development server:

npm start

The project will run locally at:

http://localhost:3000
Notes

Make sure your Node.js version is compatible.

If you encounter any issues with missing dependencies, try running:

npm install --force
